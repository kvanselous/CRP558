Exercise 12


This map is created and stored with an index.html file stored inside my exercise 12 directory in github
I used data from Dea Moines County for the locations of fire stations. Data can be downloaded here.
I used mapbox to style my base map color black, my fire stations point locations larger and red, all of my road features to white, turned off the land use feature, and made the location names green.
